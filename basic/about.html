about the library

mega-ip is a tcp/ip stack for basic and machine language

mega-ip gives the MEGA65 a small Ethernet networking layer that BASIC programs can call with SYS, while machine language programs can use the same jump table directly.

  • Ethernet, ARP, IPv4, ICMP, UDP, DHCP, DNS, and TCP live in a banked machine language runtime.
  • BASIC stays simple: configure an address, resolve a host, connect or listen, then poll and send strings.
  • The webserver demo reads static HTML files from disk and streams them through the TCP sender.
  • The goal is a reliable, well-documented stack that can support BASIC, assembly, and eventually C programs.

This about page is being served by the same BASIC webserver as the front page, selected from the browser's HTTP request path.
